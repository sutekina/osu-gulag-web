Server Switcher

Download the Server Switcher, hit Install Certificate, then Connect to Sutekina. You're done!

Certificate (only use this if switcher doesn't work for you)

  1. Download the certificate here.
  2. Open cert.der and click Install Certificate
  3. Select Current User then Place all certificates in the following store.
  4. Click Browse and select Trusted Root Certification Authorities.
  5. Click yes if prompted, the certificate installation should be succesful.

Manual Hosts Replacement (only use this if switcher doesn't work for you)

  1. Navigate to C:\Windows\System32\drivers\etc.
  2. Open the hosts file in your favorite text editor supporting administrator privileges.
  3. If you see any lines containing .ppy.sh, remove them.
  4. Add these lines to your hosts file:

    5. 212.227.215.205 osu.ppy.sh
    212.227.215.205 c.ppy.sh
